

| Tipo de objeto                       | Prefijo recomendado (2025)                            | Convención de nombre                                   | Ejemplo real                                                     | Notas importantes                                                               |
|--------------------------------------|-------------------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------|
| Tablas                               | Ninguno                                               | Singular, PascalCase                                   | Customer, Order, ProductInventory                                | Siempre singular                                                                |
| Vistas                               | Ninguno                                               | PascalCase + sufijo solo si es estrictamente necesario | CustomerOrders, ActiveEmployee                                   | La mayoría sin prefijo                                                          |
| Procedimientos almacenados           | Ninguno o usp_ solo si la empresa lo exige por legado | Verbo + Entidad + Acción (PascalCase)                  | Customer_Create, Order_Submit, Invoice_GeneratePdf               | El prefijo sp_ está desaconsejado (impacta rendimiento)                         |
| Funciones escalares                  | Ninguno                                               | Verbo + QuéDevuelve (PascalCase)                       | CalculateTax, GetFullName, DistanceBetweenPoints                 |                                                                                 |
| Funciones con valores de tabla (TVF) | Ninguno                                               | Get + Entidad + By + Filtro                            | GetOrdersByCustomer, GetActiveProducts, GetEmployeesByDepartment |                                                                                 |
| Triggers                             | Ninguno                                               | tr + Entidad + On + Evento                             | trOrder_OnUpdateInventory, trCustomer_OnDeleteAudit              | El prefijo tr sigue siendo muy aceptado porque ayuda al filtrado rápido en SSMS |
| Secuencias                           | Ninguno                                               | Entidad + Seq                                          | OrderNumberSeq, InvoiceSeq                                       |                                                                                 |
| Sinónimos                            | Ninguno                                               | Igual que el objeto remoto                             | LegacyCustomer, ExternalProductCatalog                           |                                                                                 |
| Tipos de tabla (UDTT)                | Ninguno                                               | Entidad + TableType                                    | OrderItemTableType, BulkCustomerTableType                        |                                                                                 |
| Esquemas                             | Sí (recomendado)                                      | Área funcional: Sales, Inventory, Security, Audit      | Sales.Order, Inventory.Product                                   | Muy útil en bases grandes                                                       |
